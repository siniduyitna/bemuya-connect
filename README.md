# BemuyaConnect (በሙያ ኮኔክት) 🛠️🇪🇹

**BemuyaConnect** is a high-performance Full-Stack platform designed to bridge the gap between skilled Ethiopian artisans and potential customers.

## 🏆 Performance Excellence
This platform is highly optimized for speed, accessibility, and search engine visibility. According to Google Lighthouse reports, it achieves:
* **Performance:** 100/100
* **SEO:** 100/100
* **Best Practices:** 96/100
* **Accessibility:** 90/100

---

## 🌟 Key Features
* **Artisan Directory:** A centralized hub to showcase local talents and increase artisan income.
* **Advanced Search:** District-based search functionality (e.g., Bole, Lideta) for quick expert discovery.
* **Interactive Mapping:** Powered by **Leaflet.js** for real-time visual location of service providers.
* **Bi-lingual:** Full accessibility in both **Amharic** and **English**.

---

## 🚀 Tech Stack
* **Frontend:** React.js, Vite, Bootstrap 5, Framer Motion.
* **Backend/Database:** Firebase Real-time Database.
* **Deployment:** Vercel.
